Bachelorarbeit

Entwicklung eines adaptiven Algorithmus zur Optimierung des Greifprozesses mit flexiblen, leckagetoleranten Sauggreifern zur kontrollierten Handhabung von flexiblen Bauteilen.

Zusammenfassung: Diese Arbeit befasst sich mit der Entwicklung eines lokalen Druckmesssystems und der Konstruktion funktionaler Hardwarekomponenten mittels CAD und 3D-Druck sowie der Programmierung eines Python-Algorithmus, der Leckagen in Echtzeit lokalisiert und durch autonome Ausgleichsbewegungen korrigiert.

Abstract lesen

▶ CAD-Konstruktion

Konstruktion in Siemens NX & Solid Edge, anschließend 3D-gedruckt.

▶ CFD-Simulation

Strömungssimulation in Inkscape/Ansys zur Analyse des Ansaugverhaltens.

▶ Python Steuerungs-Code

Auszug aus der Leckageanalyse und automatischen Roboter-Korrektur:

# Auszug: Bewertung des Griffstatus anhand der vier Drucksensoren
def check_grip_success(plc, ur_controller, current_setpoint):
    val1 = plc.read_by_name("GVL.P_in[1]") + 1
    val2 = plc.read_by_name("GVL.P_in[2]")
    val3 = plc.read_by_name("GVL.P_in[3]") - 1
    val4 = plc.read_by_name("GVL.P_in[4]")
    values = np.array([val1, val2, val3, val4])

    avg_p    = np.mean(values)
    max_diff = np.max(values) - np.min(values)
    p_ref    = get_reference_pressure(current_setpoint)
    dev_ref  = np.max(np.abs(values - p_ref))

    # Fall 1: Fehlversuch (kein Vakuum aufgebaut)
    if avg_p > -1.0 and np.all(values > -3.0):
        return "Failed", values, None

    # Leckage-Sensor identifizieren
    leak_indices         = np.where(values == np.max(values))[0]
    primary_leak_sensor  = leak_indices[0] + 1

    is_homogenous = max_diff <= 2.0
    is_near_ref   = dev_ref  <= 4.0

    # Fall 2: Vollständiger, sauberer Griff
    if is_homogenous and is_near_ref:
        return "Success", values, None

    # Fall 3a: Mittige Leckage  /  Fall 3b: Seitliche Leckage
    if is_homogenous and not is_near_ref:
        return "Leakage Central", values, primary_leak_sensor
    return "Leakage Side", values, primary_leak_sensor
