Solarapp — Kurzfassung

// Frederik Hoeft  ·  Persönliches Software-Projekt

Server-Anwendung zum lokalen Auslesen eines Photovoltaik-Hybridwechselrichters mit Live-Monitoring, historischer Datenanalyse und Wirtschaftlichkeitsberechnung.

Kurzfassung (Deutsch)

Solarapp ist eine eigenständig entwickelte Software zur Überwachung und wirtschaftlichen Auswertung einer privaten Photovoltaikanlage mit Batteriespeicher. Anders als herstellergebundene Cloud-Lösungen liest die Anwendung den Deye-Hybridwechselrichter (SUN-8K-SG01HP3) vollständig lokal aus — ohne Internetdienst, datenschutzfreundlich und unabhängig von der Verfügbarkeit externer Server.

Die App wird als Server auf Basis von Python (FastAPI/Uvicorn) betrieben und kommuniziert im selben Netzwerk über den Solarman-Datenlogger per Modbus/TCP (Port 8899) mit dem Wechselrichter. Der Zugriff erfolgt ausschließlich lesend (read-only); es werden zu keinem Zeitpunkt Werte in den Wechselrichter geschrieben. Die erfassten Messwerte werden in einer SQLite-Datenbank gespeichert, sodass die Datenbasis auch dann lückenlos bleibt, wenn der Rechner nicht durchgehend läuft.

Das Web-Dashboard wird ohne Build-Prozess direkt von der Anwendung ausgeliefert (Chart.js und Alpine.js) und gliedert sich in drei Ansichten: eine Live-Übersicht über PV-Leistung, Hausverbrauch, Netzbezug und Batteriestand inklusive Autarkiegrad; eine Verlaufsansicht mit Tages-, Monats- und Jahresdiagrammen für Ertrag, Verbrauch, Einspeisung, Netzbezug und Eigenverbrauch; sowie eine Ansicht zur Wirtschaftlichkeit.

Die Wirtschaftlichkeitsberechnung basiert auf den kumulativen Zählern des Wechselrichters und ermittelt pro Jahr die Stromkostenersparnis, den Einspeise-Erlös sowie die Eigenverbrauchs- und Autarkiequote. Auf dieser Grundlage prognostiziert die App die Amortisation der Anlage einschließlich des Break-even-Zeitpunkts.

Solarapp verbindet damit Echtzeit-Monitoring, langfristige Datenhaltung und eine transparente Wirtschaftlichkeitsbetrachtung in einer schlanken, lokal gehosteten Anwendung und macht den tatsächlichen Nutzen der Photovoltaikanlage jederzeit nachvollziehbar.

Abstract (English)

Solarapp is a self-developed software application for monitoring and economically evaluating a private photovoltaic system with battery storage. Unlike manufacturer-bound cloud solutions, the application reads the Deye hybrid inverter (SUN-8K-SG01HP3) entirely locally — without any internet service, in a privacy-friendly manner and independently of the availability of external servers.

The app runs as a server based on Python (FastAPI/Uvicorn) and communicates with the inverter over the same network via the Solarman data logger using Modbus/TCP (port 8899). Access is strictly read-only; no values are ever written to the inverter. The acquired measurements are stored in a SQLite database, so that the data set remains gap-free even when the computer is not running continuously.

The web dashboard is served directly by the application without any build step (Chart.js and Alpine.js) and is divided into three views: a live overview of PV power, household consumption, grid draw and battery level including the self-sufficiency rate; a history view with daily, monthly and yearly charts for yield, consumption, feed-in, grid draw and self-consumption; and a profitability view.

The profitability calculation is based on the inverter's cumulative meters and determines, for each year, the savings on electricity costs, the feed-in revenue as well as the self-consumption and self-sufficiency rates. On this basis, the app forecasts the system's amortisation, including the break-even point.

Solarapp thus combines real-time monitoring, long-term data storage and a transparent profitability analysis in a lean, locally hosted application, making the actual benefit of the photovoltaic system traceable at any time.
